AUDIO MORSE :: MICROPHONE

—

Tap LISTEN, then play steady beeps. Short=dot, long=dash. Auto-decodes after a pause. Needs mic permission (HTTPS).

LIGHT MORSE :: CAMERA

—

Point rear camera at a blinking light / torch. Set threshold above ambient so the bar fills only on flashes. Needs camera permission (HTTPS).

MANUAL MORSE :: TEXT
